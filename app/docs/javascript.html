Basic Structure

function control(car) {
    // Your code
}

Controls

car.steering: Number: Steering direction. -1 for left, 1 for right
car.acceleration: Number: Acceleration. 1 for full forward, -1 for full reverse
car.brake: Number: Break. 1 for full braking, 0 for no braking
car.nitro: Number: Nitro boost. 1 for full boost, 0 for no boost
car.message: String: Debugging message, displayed at top

Sensors

car.speed: Number: Current car speed
car.bearing: Number: Current car bearing relative to the car's forward. Close to 0 except when drifting.
car.obstacle_detection(start, end[, cap]): Number: Distance to nearest obstacle between the angles of start and end. cap can be used to reduce the detection range. Returns api.NO_DETECTION if no obstacles are near enough.
car.obstacle_detection_rays: Object[Number, Number]: Dictionary of angle to distance to nearest obstacle at that angle
car.obstacle_detection_range(angle): Number: Maximum range of obstacle detection rays for that angle
car.waypoint_distance: Number: Distance to next waypoint
car.waypoint_bearing: Number: Bearing to next waypoint relative to the car's forward
car.future_waypoint_distance: Number: Distance to waypoint after the current one
car.future_waypoint_bearing: Number: Bearing to waypoint after the current one relative to the car's forward

Notes

console.log outputs to the console, so you will need to open your dev tools so view your script's output. Using car.message may be better.
