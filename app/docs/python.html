Controls

car.steering: float: Steering direction. -1 for left, 1 for right
car.acceleration: float: Acceleration. 1 for full forward, -1 for full reverse
car.brake: float: Break. 1 for full braking, 0 for no braking
car.nitro: float: Nitro boost. 1 for full boost, 0 for no boost
car.message: str: Debugging message, displayed at top

Sensors

car.speed: float: Current car speed
car.obstacle_detection(start, end[, cap]): float: Distance to nearest obstacle between the angles of start and end. cap can be used to reduce the detection range. Returns api.NO_DETECTION if no obstacles are near enough.
car.obstacle_detection_rays: dict[float, float]: Dictionary of angle to distance to nearest obstacle at that angle
car.obstacle_detection_range(angle): float: Maximum range of obstacle detection rays for that angle
car.waypoint_distance: number: Distance to next waypoint
car.waypoint_bearing: number: Bearing to next waypoint
car.future_waypoint_distance: number: Distance to waypoint after the current one
car.future_waypoint_bearing: number: Bearing to waypoint after the current one

Notes

print outputs to the console, so you will need to open your dev tools so view your script's output. Using car.message may be better.
This uses Brython which is a close implementation of Python 3.5. There are, however, some irregularities so if your code fails try simplifying the syntax.
