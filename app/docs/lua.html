Basic Structure

function control(car)
    -- Your code
end

Sensors

car.speed: Current car speed
car.danger: Obstacle in front of the car

Controls

car.forward: Go forward
car.backward: Go backward
car.left: Go left
car.right: Go right

Notes

print outputs to the console, so you will need to open your dev tools so view your script's output.
